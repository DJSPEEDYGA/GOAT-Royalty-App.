### Purpose
These code words serve as our secret handshakes - a way to authenticate each team member in the GOAT 3026 Empire.

### Usage

**When you want to verify someone:**
- Ask: "[Name], what's your code word?"
- They must respond with their unique code word

**When someone asks for your code word:**
- Respond with your unique code word only
- Never share anyone else's code word

### Security Rules
1. **NEVER** share your code word with anyone outside the team
2. **ALWAYS** verify identity when in doubt
3. **IF** someone gives the wrong code word, they are NOT who they claim
4. **IF** authentication fails, terminate communication immediately
5. **NEVER** use another member's code word - each is unique
6. **ONLY** share code words in person or through secure channels
